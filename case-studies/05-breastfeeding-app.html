01

Context

Project Background

母乳哺育對媽媽及嬰兒來說有許多的優點，但是仍有以下問題： -（一） 我國2016年的6個月內母乳哺育率為50% 雖然已經有半數，但仍有進步的空間。事實上，要提升母乳哺育率，可能要讓孕婦從懷孕期就開始做準備，包括哺乳育兒計畫以及醫院完整衛教計畫，都應在早期開始，以建立孕婦未來的衛教知識及自信心。 -（二） 現在早期的衛教計畫，多數以口頭、書面及影音方式的「教導方式」為主 因為沒有什麼機會實際操作，很難讓孕婦建立長久的衛教知識和信心。 -（三） 未來哺育行為不一定能夠持續 孕婦因為衛教方面知識的缺乏，造成沒有信心，使得很難讓未來的哺育行為能夠持續，並降低配方奶/牛奶使用的機會。 媽媽餵為國內母乳哺育相關產品的龍頭公司之一，長久以來，對於母乳哺育的推動遺餘力，包括自家研發方便母乳哺育的相關產品，如月亮枕、吸乳器、哺乳衣…等，媽媽餵對於衛教的施行亦貢獻良多，包括媽媽教室的設立以及媽媽哺乳育兒小百科的建置等。然而，以廠商的角度研發哺育產品及線上衛教知識，無非是希望達到消費者及生產者雙贏，也就是潛在顧客（懷孕的媽媽）能長期使用衛教IT產品（例如衛教app），並黏著於衛教app產品上（例如1星期登入2次使用），如此潛在顧客日後才有機會持續哺乳育兒至少6個月，並且日後才有機會能「樂於且自然」使用相關產品解決自身痛點（例如奶水不足使用吸乳器），以進而達到雙 贏，也就是潛在顧客獲得持續餵母乳的好處，廠商亦能獲得合理的利潤。 然而，要達到此雙贏局面，必須在初期即吸引潛在顧客（懷孕媽媽）使用衛教app，以建立早期的哺乳育兒計畫，並增進衛教知識以獲得自信心，因此衛教app的設計相當重要。

Target Users

新手父母

02

Research & Insights

Research Methods

  • Persona

小雅是三十歲和老公住在台中的懷孕媽媽，在產檢時看到醫院張貼關於親餵母乳的信息，加上身邊朋友也有相關經驗，所以對餵母乳產生好奇並想等寶寶出生後進行母乳哺餵。 平日除了上班，小雅還須利用空餘時間蒐集相關衛教資料，但網路上的資訊過多，一些其他媽媽們的經驗分享又缺乏實證，不知道什麼是可以放心參考的。雖然產檢時可以順便詢問醫生，但來回的不斷確認讓小雅感到疲憊與焦慮…

  • Desktop research

我們將其大致統整成三個大點，分別是衛教資訊不足、缺乏專業協助和可能產生的憂慮等類別。 首先，衛教資訊不足的部分涵蓋因哺乳相關知識不足進而開始擔心會收集到錯誤資訊、擔心哺乳姿勢不正確和不知道什麼食物該吃、什麼不該吃等等的疑慮。 第二，缺乏專業協助的部分會有的痛點則是因專業人員的難尋性所引發的，像是媽媽們可能會為此感到有點焦慮，怕日後出了什麼料想不到的情況，但卻找不到專業人士可以從旁協助、引導他們解決問題。 最後，媽媽們對於餵母乳這個行為本身就會產生很多的擔憂，像是怕家人不支持、怕睡眠時間和品質被壓縮、怕乳腺炎和塞奶問題嚴重影響自身和怕沒有實際餵過母乳的自己會做得不好、對自己沒信心等多個痛點。 我們則主要是以餵母乳對媽媽的影響及對寶寶的影響兩個方向來做區分。 首先，對媽媽的影響包含像是能降低罹患乳癌、卵巢癌等疾病的機率、省錢、提升 媽媽與寶寶的親密度、減少產後大出血和維持身材等多項益處。 再來，對寶寶的影響則是有能夠增加免疫力、促進寶寶腦部神經發育、營養價值高和過敏風險低等多個促使媽媽們最後決定克服萬難餵母乳的關鍵甜蜜點。

  • 關鍵字心智圖

我們根據痛點與甜蜜點，整理出相關的關鍵字，並利用awoo SEO網站分析出以下關鍵字的搜尋量。 從心智圖中我們可以看到，對媽媽的影響中，「內衣」搜尋量最多，「月經」次之，可見媽媽特別在乎哺乳時要穿什麼內衣，以及哺乳期對月經來潮的影響。 而對寶寶的影響中，我們可以發現「配方奶」的搜尋量最高。顯示多數媽媽仍有考慮餵配方奶，不過也有相當多人搜尋「母乳跟配方奶」的比較，還有「母乳營養」，因此如果想讓媽媽們選擇餵母乳，可以從強調母乳跟配方奶的營養差異著手；而「親餵」的搜尋量遠遠大於「瓶餵」，顯示選擇餵母乳的媽媽多數會在親餵的狀況下有相關問題。

  • 情境流程圖

小雅今年30歲，剛初次懷孕滿三個月。平日要忍著頭暈、噁心等種種身體狀況上班，假日又要為了選擇對寶寶最好的哺育方式上網辛苦查資料。但對她來說最痛苦的是網路上的資訊非常雜亂且很多言論都未經專家證實，常將她搞得一頭霧水，不知道究竟哪個說法具有參考價值。而若轉問身邊有經驗的親友，大多也因各自面臨的狀況不同，無法給小雅準確地回答。產檢也因為是一個月一次，遇到問題也無法即時請教醫生。小雅因此感到手足無措，想給寶寶最好的營養，但又因為缺乏相關知識和資訊不知道怎麼做…

  • 情境分鏡圖
  • 情境一：

懷孕三個月時，平日除了上班，假日還要在家裡煩惱著等孩子出生後究竟是要餵母乳還是泡配方奶，也要上網搜尋各種哺育及其他懷孕媽媽的相關心得文章。此時，若是有一個諮詢平台，讓小雅能即時詢問想問的問題，就能有效減緩其身、心靈所承受的壓力。 →懷孕了既要想要餵配方奶還是母乳，又要查衛教資訊很麻煩

  • 情境二：

小雅上網找資料時，不同網站的說法都眾說紛紜，什至就算加入大多已有經驗的媽媽社團後，還是會看到各式各樣不同、什至錯誤的衛教資訊，讓小雅很困擾。因此，若是有專門讓媽媽發問的平台且回答問題的都是受過特定訓練的專業人士，就能大大降低錯誤衛教資訊傳播的可能性，也能縮短小雅查詢相關資訊的時間。 →網站資訊量過大又不一定正確

  • 情境三：

由於自身對餵母乳相關知識的缺乏，小雅會去詢問身邊親友，請他們分享經驗。但畢竟每個人遇到的狀況不同，如果是一些會直接或間接危害到寶寶健康的專業問題，小雅還是必須向專業人士請教，做二次確認才能安心。對一個懷孕三個月還要工作的婦女來說，實在是勞心又勞身。此時，若有東西能像百科全書一樣將正確資訊都整理好集中在一起就能幫她減輕許多負擔。 →問親友每個人遇到的狀況不同

  • 情境四：

小雅每天都要頂著肚子在公司工作，大部分的時間都需要坐在椅子上，而這一行為會使她的脊椎感到不適，且工作期間也偶爾會有頭暈、噁心等孕吐現象。此時，孕吐方面的問題還需靠平時透過少量多餐的方法改善，降低工作中想嘔吐的可能性；至於工作任務負擔方面，則是可以依《勞動基準法》向公司申請改調，以減緩工作所帶來的身心靈負擔。 →懷孕中還要工作，身體負擔重

  • 情境五：

懷孕三個月後，小雅偶爾產檢時間會排在平日。這時，她可以向公司請帶薪產檢假。然而在產檢前也是需要事先上網查詢相關資訊，但小雅發現這些資訊不僅多是大同小異，還時常有很多明顯是收錢辦事的水軍所寫的評論。導致小雅常需要來回確認資訊的真偽。這時，若能創建一個具權威的相關平台讓醫師、護理師等具備相關專業知識的人做正確的資訊傳遞，就能降低小雅的焦慮感。 →產檢久久一次，有問題也來不及請教醫生

Key Findings / Main Problems

  • 一、 功能性

設計此一衛教知識APP旨在替媽媽們省下大量上網查詢母乳相關資訊的時間及降低搜尋到錯誤資訊的可能性，主要提供經專業人士確認過的衛教影片，並以遊戲化設計的方式包裝，為原本枯燥乏味的學習增添許多趣味性。像是媽媽們可以在觀看完衛教影片後到「知識王」區去和其他使用者以遊戲PK形式互相切磋看看誰的衛教方面知識更勝一籌，而每次挑戰完後勝者可以獲得一定數量的金幣。金幣在累積到特定數量後能解鎖髮型、衣服或褲子等等，讓媽媽去客製化自己或寶寶的人物形象，營造個人化專屬感。

  • 二、 情緒性

發現媽媽們不論是身處懷孕或哺乳階段，若能有其他同伴的相互支持和陪伴，對她們來說絕對會是一大助力，也會提高她們自信心，堅定自己能持續哺乳。因此，我們也在APP內特意打造了一個專屬媽媽們的社群，讓他們能相互交流、分享心情。而這個社群和臉書上的媽媽社團的最大不同之處在於媽媽們的每個困惑、疑問會由受過專業訓練的媽媽餵專員來做回覆，不會再有媽媽們被各方說法搞得一頭霧水的情況發生。

Key Insights

  • 如果 APP 能快速傳遞給媽媽們正確的衛教知識，不僅能順利化解大部分的痛點，還能大幅提升媽媽們願意親餵母乳的意願，並進而提升媽媽餵品牌辨識度和各式商品買氣。

  • 使用者行為分析（Jobs-to-Be-Done, JTBD） 量性分析

機會點由高到低的結果（機會點分數越高，則代表越有市場性）

  • - 在醫療院所餵母乳時，降低因為知識不足，未來親餵母乳時有錯誤知識的發生[分數:12.69]
  • - 在醫療院所餵母乳時，減少嬰兒一直哭（想吃奶），而產生疲憊感[分數:11.92]
  • - 在醫療院所餵母乳時，增加護理人員實際示範餵母乳的機會[分數:10.77]
  • - 在醫療院所餵母乳時，增加未來能長久親餵母乳的自信心[分數:9.23]
  • - 在醫療院所餵母乳時，增加親餵母乳次數，而與嬰兒有高親密感（低疏離感）[分數:9.23]
  • - 在醫療院所餵母乳時，降低懷疑未來是否能成功親餵母乳的機會[分數:8.85]
  • - 在醫療院所餵母乳時，強化初餵母乳時的驚喜感（例如一出生就會吃）[分數:8.46]
  • - 在懷孕時，強化醫療院所提供哺乳計畫（讓妳提早做準備）[分數:7.69]
  • - 增加未來家人及親友支持親餵母乳[分數:5.77]
  • - 在醫療院所餵母乳時，降低因為怕哺乳能力不足，會想加配方奶的機會[分數:5]
  • - 在醫療院所餵母乳時，降低未來想改成餵配方奶的想法發生的機會[分數:5]

許多潛在顧客或文獻覺得重要的痛點，可能潛在顧客已經有自己滿意的解決方案，因此機會點不高（市場潛力），若取機會點7以上的結果（根據結果，機會點在7.5為一個門檻），我們發覺潛在顧客最大的市場需求是不要有錯誤知識的發生，即使市場上已經有相關的衛教app或網站產生，但是可能這些網站及app並無法滿足其需求，潛在顧客沒辦法持續黏著在這些app或網站上面，潛在顧客可能還是需仰賴網路上同儕的親身經驗，但這些親身經歷可能未有實證研究支持，或者是不一定值得信賴的業配文知識。 因此，我們以遊戲化方式設計一個衛教app有其適當性，可以讓潛在顧客持續黏著在正確且值得信賴的哺乳知識上，以強化其未來的哺乳知識、態度及行為的改變，此外，根據JTBD調查結果，我們發覺遊戲化衛教的內容可以包括:嬰兒一直想哭時的策略、正確示範親餵母乳、如何增加初期親餵母乳的自信心，以及初餵母乳的親密感及驚喜感、提供未來哺乳計畫等，這些需要內容發生的場景，可能包括未來在家、工作哺育時以及在共種場合時。

Key Design Decisions

  • 遊戲化衛教App設計原理

我們預想懷孕媽媽會經過三個階段使用，包括搜尋階段（Discovery）、上手階段（Onboarding），以及支撐黏著階段（Scaffolding），有關於三個階段的重點設計，以及如何利用遊戲化八角框架設計，請見下。

  • - 搜尋階段（Discovery）

這個階段著重的是讓懷孕媽媽接觸到媽媽餵的任何資訊後（非此次比賽範圍），能夠上網站看看，並且註冊且試用。 我們期望懷孕媽媽有如下的預期行為

  1. 到媽媽餵網站
  2. 媽媽餵網站學習及互動

（1）讀媽媽百科 （2）報名媽媽教室 （3）看使用者見證

  1. 嘗試試用版的媽媽餵遊戲化衛教
  2. 註冊相關的資料

相關的八角框架驅動力原理請見括弧內文字，這些驅動力能讓媽媽完成前述的預期行為

  1. 寶貝的健康不能等，妳願意為台灣寶貝的哺育健康盡一份心力嗎? （標語-史詩般招喚）
  2. 初階拖拉式的知識遊戲測試哺乳相關小知識 （發展與成就）
  3. 首頁的益智遊戲，通過後註冊即能領取獎勵 （創意與培力）
  4. 好的哺乳衛教工具，是值得你花時間投資的 （標語-擁有與占有）
  5. 一年兩次折扣，提供給一次訂閱一年（每個月XXX台幣）的懷孕媽媽（稀有性）
  6. 可以看到已拿到證書的資深媽媽（稀缺性）
  • - 上手階段（Onboarding）

這個階段著重的是讓懷孕媽媽能夠選擇6個面向成長的知識，並且持續誘導媽媽學習，讓懷孕媽媽能了解自己及他人的學習進度，同時希望懷孕媽媽能使用媽媽社群，並且遵循建議。 我們期望懷孕媽媽有如下的預期行為

  1. 學習如何增長哺乳遊戲實力值（XP）
  2. 選擇6個成長路徑（懷孕篇、醫院篇、上班篇、外出篇、在家篇、飲食篇）
  3. 看每個路徑的衛教建議影片
  4. 對衛教影片留評論
  5. 登錄媽媽社群成為會員
  6. 遵循高階媽媽會員的一些建議
  7. 對社群介紹妳自己（包括自己的背景，以及想要在媽媽餵遊戲化衛教app達到什麼）

相關的八角框架驅動力原理請見括弧內文字，這些驅動力能讓媽媽完成前述的預期行為

  1. 妳被選出來證明，媽媽餵遊戲化衛教app可以改變妳的哺育生活，過得更好（標語-史詩般招喚）
  2. 會員PK（知識王形式，能檢測自身的衛教知識程度，增加XP）（發展與成就）
  3. 顯示各成長路徑的增長實力值 （發展與成就）
  4. 隨機三個每日任務video （發展與成就）
  5. 成長路徑雷達圖會隨著知識增長而改變各面向形狀 （創意與培力）
  6. 挑戰哺乳相關知識，以獲得獎勵 （創意與培力）
  7. 懷孕媽媽選擇學習路徑，客製化學習路徑且學習完成後可獲得認證證書 （創意與培力）
  8. 說明某位懷孕媽媽有幾個FB朋友也加入會員 （社群與關係）
  9. 打開每日的限時問題箱子（每日小測驗），以獲得coins （稀缺性）
  10. 打開彩蛋箱，可以獲得撫育幼兒小道具（神秘感與不可預測性）
  11. 有參加親餵教學工作坊的機會，在時間內報名才有discount（避免損失性）
  • - 支撐階段（Scaffolding）

這個階段的重點在於如何讓懷孕媽媽有黏著性，讓懷孕媽媽能持續登入、持續收集金幣、撫育虛擬寶寶、以及與成功哺乳的資深媽媽互動。 我們期望懷孕媽媽有如下的預期行為

  1. 每周登入兩次
  2. 看新的且有興趣的內容
  3. 收集金幣（一天1~2次）
  4. 檢視各成長路徑的dashboard，並得知與下一級差距多少
  5. 與成功哺乳的媽媽夥伴/社群對談
  6. 撫育虛擬寶寶，金幣購買媽媽餵虛擬商店商品

相關的八角框架驅動力原理請見括弧內文字，這些驅動力能讓媽媽完成前述的預期行為

  1. 懷孕媽媽可以選擇金幣多少百分比（%），可以轉換成現金，捐給哺乳公益團體（標語-史詩般招喚）
  2. 懷孕媽媽，在玩虛擬撫養寶寶遊戲時，可以客製化自己以及寶寶的角色形象（創意與培力）
  3. 每次懷孕媽媽完成一個單元（有許多問題集），會有寶物箱收集coins及撫育道具 （擁有與占有）
  4. 根據懷孕媽媽的偏好，在某個學習路徑上，建議他們最相關的影片觀看（擁有與占有）
  5. 在自己blog寫使用經驗文章（例如泌乳器使用文），可換取金幣（擁有與占有）
  6. 每天會算出懷孕媽媽進入十大排行榜的排名（有將近100種排行榜，例如這一周的飲食路徑看最多影片者），然後會顯現給懷孕媽媽看，不會出現神人霸榜（前幾名都是固定幾位），懷孕媽媽每次看榜都能覺得有成就（發展與成就）
